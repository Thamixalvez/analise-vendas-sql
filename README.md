# 📊 Análise de Dados com SQL

Projeto de análise de dados utilizando SQL para explorar informações de um conjunto de dados financeiros.

---

## 🛠 Tecnologias utilizadas

- SQL
- CSV
- Visual Studio Code

---

## 📂 Estrutura do projeto

analise-vendas-sql
│
├── consultas_sql
│   └── analise_vendas.sql
│
└── dados
    └── Financeiros.csv

---

## 📊 Análises realizadas

- Total de valores registrados
- Quantidade de registros por categoria
- Soma de valores por categoria
- Top 5 maiores valores

---

## 🚀 Como usar

1. Abra o arquivo `analise_vendas.sql`
2. Utilize o dataset localizado em `dados/Financeiros.csv`
3. Execute as consultas em um banco SQL (SQLite, MySQL ou PostgreSQL)
